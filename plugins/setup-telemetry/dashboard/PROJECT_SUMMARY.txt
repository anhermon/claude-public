================================================================================
API CONTRACTS
================================================================================

GET /api/sessions?page=1&pageSize=20&status=active&userId=user-123
└─ Returns: SessionListResponse { sessions[], total, page, pageSize, totalPages }

GET /api/sessions/[id]
└─ Returns: SessionDetailResponse { session }

GET /api/metrics?fromDate=1234567890&toDate=1234567890
└─ Returns: MetricsResponse { KPIs, toolMetrics[], errorBreakdown[], sessionsByHour[] }

GET /api/live
└─ Returns: LiveFeedResponse { events[], activeSessionCount }

All endpoints return no-cache headers (Cache-Control: no-store, must-revalidate)
